Описание проекта
Movie Library — это графическое приложение на Python для управления личной коллекцией фильмов. Позволяет добавлять фильмы с проверкой данных (год и рейтинг), просматривать их в таблице и фильтровать по жанру или году. Все данные сохраняются в файл movies.json, что позволяет не терять коллекцию после закрытия программы.

Возможности
Добавление фильмов: название, жанр, год выпуска (от 1888 до 2026), рейтинг (от 0 до 10).
Валидация ввода: год — только число в допустимом диапазоне; рейтинг — число от 0 до 10.
Фильтрация по жанру и году.
Сохранение и загрузка данных из movies.json.
Интуитивно понятный интерфейс на Tkinter.

Требования
Python 3.6+
Библиотека Tkinder (входит в стандартную поставку Python).

Инструкция по запуску
1. Убедись, что установлен Python 3.6 или новее.
2. Склонируй репозиторий:
git clone https://github.com/[твой_логин]/Movie-Library.git
cd Movie-Library
3. Запусти приложение:
python main.py
4. Для работы с Git используй стандартные команды (git add ., git commit -m "...", git push).

Примеры использования
Добавить фильм: заполни все поля и нажми «Добавить фильм».
Фильтровать по жанру: введи жанр и нажми «Фильтровать».
Фильтровать по году: введи год и нажми «Фильтровать».
Данные сохраняются автоматически при добавлении фильма.
